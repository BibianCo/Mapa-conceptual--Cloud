##  Mapas Conceptuales – Computación en la Nube

Este repositorio contiene los mapas conceptuales elaborados para la actividad sobre computación en la nube. Cada mapa resume de forma clara los conceptos principales relacionados con los tipos de nubes, los modelos de servicio y los proveedores más utilizados actualmente.

  Temas incluidos
## 1. Tipos de Nube

   Nube Pública
   Nube Privada
   Nube Híbrida
   Nube Comunitaria

Cada una se presenta con una breve explicación sobre sus características y usos.

## 2. Modelos de Servicio en la Nube

IaaS: Infraestructura como servicio

PaaS: Plataforma como servicio

SaaS: Software como servicio

Se muestran las diferencias entre ellos y ejemplos de uso.

## 3. Proveedores de Servicios en la Nube

Amazon Web Services (AWS)

Microsoft Azure

Google Cloud Platform (GCP)

Oracle Cloud Infrastructure (OCI)

Incluye ejemplos representativos de los servicios que ofrece cada proveedor.
